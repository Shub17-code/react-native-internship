# Day 6 - Styling in React Native 🎨

## Topics Covered

- Inline Styling
- StyleSheet
- Basic Layout (Flexbox)

## What I Learned

- Styles in React Native are written in JavaScript
- StyleSheet helps organize styles cleanly
- flex: 1 is used to take full screen space
- justifyContent aligns items vertically
- alignItems aligns items horizontally

## Code Concepts Used

- StyleSheet.create()
- Applying styles using style={}
- Centering UI elements

## Output

- Styled screen with title, subtitle and button
